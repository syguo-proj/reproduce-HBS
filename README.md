# Reproduction of Hamming Ball Sampler

This repository reproduces the Hamming Ball Sampler (Titsias & Yau, 2017), an MCMC sampling procedure for high-dimensional discrete status space exploration.

## Contents

- reproduce_HBS.Rmd: Includes main functions for hamming ball generating, MCMC sampling procedure, and experiments on high-dimensional linear regression variable selection.
- simulation-report-HBS.pdf: Includes a brief review for HBS methodology and simulation results illustrated via the trajectory of posterior inclusion probabilities over iterations.
